ionic入门（二）-环境搭建

上一篇文章《ionic入门（一）-简述》为大家简单介绍了学习ionic以及angularjs的相关资料，从这些资料中应该了解到ionic的一些知识。本篇文章将教大家如何搭建ionic的开发环境，其实也是没什么难度，看来文档应该也会搭建的了, 分成几个步骤：

  1. 配置Java环境变量
  2. 下载安装NodeJS
  3. 配置Android环境变量，并下载SDK
  4. 安装Cordova、ionic

下面跟着步骤来做吧:

注意： 本篇环境配置是基于Windows环境的，Mac环境自行上网搜索下，步骤基本差不多

1. 配置Java环境变量

JAVA SDK下载推荐大家到oracle官网按需要下载对应版本: jdk-8u73-windows-x64.exe

下载后正常安装，记得JDK的安装路径,环境变量配置需要它。

安装完成后直接快捷键win+x调出菜单(win7开始菜单>计算机>右键) > 系统 > 高级系统配置 > 环境变量，然后在系统变量里面添加以下变量：

  • JAVA_HOME = [JDK安装路径]，如D:\DevTools\Java\jdk1.8.0_73
  • CLASSPATH = .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; ，注意前面.和后面;。

然后在 PATH 变量最后加入： %JAVA_HOME%\bin;。 win10在PATH追加内容更方便，但是win10上不需要分号。

确定关闭，开启cmd验证，全局输入下命令：

2. 下载安装NodeJS

这个没什么好说的，大家到官网下载对应版本直接安装就好了，它安装过程会自动添加环境变量的，安装完使用cmd验证下就好了：

node -v
npm -v

打印对应版本信息即安装成功

3. 配置Android环境变量

首先大家先下载Android的SDK，然后放对相关目录里下，如D:\DevTools\android-sdk-windows。

SDK国内下载地址： http://www.androiddevtools.cn/

先下载SDK Tools，解压到相关目录，然后再下载SDK Platform-Tools、Build-Tools、SDK等，sdk可以先不下载，按照上面网站的提示解压到对应路径。

下载解压完后在环境变量里面配置以下属性：

  • ANDROID_HOME = D:\devTools\android-sdk-windows

然后在 PATH 变量最后加入： %ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools;。 cmd输入android弹出sdk管理器即配置成功。

4. 安装Cordova、ionic

最后安装node modules，即cordova 和 ionic，使用cmd输入命令：

npm install -g cordova， 全局安装cordova
npm install -g ionic， 全局安装ionic

可能国内npm下载会很慢，大家可以尝试淘宝cnpm镜像加速： https://npm.taobao.org/

同样安装完后键入命令验证即可：

cordova -v
ionic -v

另外，推荐大家把gulp和bower也安装了，后面前端资源管理需要用到他们：

npm install -g gulp
npm install -g bower
